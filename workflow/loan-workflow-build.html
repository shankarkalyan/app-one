Loan Assumption Process Flowchart

Complete Flow with all paths

Process Step
Decision
SQ Review
Denial
Loan Closed
flowchart TD
    A[Call received from Customer or Authorized 3rd Party]
    B[Agent asks questions in Case Optimizer]
    A --> B

    C{Decide if assumption is allowed}
    B --> C

    D[Send Ineligible Letter]
    C -- No --> D
    END1((End))
    D --> END1

    F[Send Application]
    C -- Yes --> F

    G[DocuSign]
    F --> G

    H[Application Returned]
    G --> H

    I[Review Uploaded Package]
    H --> I

    J{Is Complete or less than 25 days}
    I --> J

    K[Send Incomplete Closure Letter]
    J -- No --> K
    END2((End))
    K --> END2

    SD[Create and Send Disclosure Package]
    J -- Yes --> SD

    SQ1[SQ Review]:::sqStyle
    SD --> SQ1
    SQ1 -- Fail --> SD

    ALR[Application gets Assigned for Loan Review]
    SQ1 -- Pass --> ALR

    DOCCHECK{Documents needed?}
    ALR --> DOCCHECK

    CML[Create Letter for Missing Documents]
    DOCCHECK -- Yes --> CML

    SQ2[SQ Review]:::sqStyle
    CML --> SQ2
    SQ2 -- Fail --> CML

    SLC[Send Letter to Customer]
    SQ2 -- Pass --> SLC

    DR[Documents Returned]
    SLC --> DR
    DR -- Received --> ALR

    CW[Customer Withdrawn]
    DR -- Not Received --> CW
    END3((End))
    CW --> END3

    UWC[Underwriting Checklist]
    DOCCHECK -- No --> UWC

    AU[Assigned to Underwriter]
    UWC --> AU

    URC[Underwriter Review for Completeness]
    AU --> URC

    READY{Ready?}
    URC --> READY
    READY -- Not Ready --> ALR

    DEC{Make Yes or No Decision}
    READY -- Ready --> DEC

    MCL[Make Commitment Letter]
    DEC -- Yes --> MCL

    SQ3[SQ Review]:::sqStyle
    MCL --> SQ3
    SQ3 -- Fail --> MCL

    SL[Send Letter]
    SQ3 -- Pass --> SL

    ACA[Assigned to Call Agent]in pl
    SL --> ACA

    RWA[Review With Customer]
    ACA --> RWA

    CCP[Create and Complete Closing Packet]
    RWA --> CCP

    SQ4[SQ Review]:::sqStyle
    CCP --> SQ4
    SQ4 -- Fail --> CCP

    SCP[Send Closing Packet to Title Agency]
    SQ4 -- Pass --> SCP

    RCP[Review Closing Packet]
    SCP --> RCP

    SQ5[SQ Review]:::sqStyle
    RCP --> SQ5
    SQ5 -- Fail --> RCP

    CSM[Complete System Maintenance - MSP]
    SQ5 -- Pass --> CSM

    SQ6[SQ Review]:::sqStyle
    CSM --> SQ6
    SQ6 -- Fail --> CSM

    END4((End - Loan Closed)):::successStyle
    SQ6 -- Pass --> END4

    MDL[Make Denial Letter]:::denyStyle
    DEC -- No --> MDL

    SQ7[SQ Review]:::sqStyle
    MDL --> SQ7
    SQ7 -- Fail --> MDL

    SLD[Send Denial Letter]
    SQ7 -- Pass --> SLD

    END5((End - Denied))
    SLD --> END5
