1 Getting Started - Create a Loan Application

Begin by simulating a new loan assumption application to see the workflow in action.

Navigate to New Application

  1. Open your browser and go to http://localhost:5173
  2. You'll see the Applications List page
  3. Click the "+ New Application" button in the top right

Fill in Application Details

  1. Borrower Name: Enter the new borrower's full name (e.g., "John Smith")
  2. SSN Last 4: Enter the last 4 digits of SSN (e.g., "1234")
  3. Loan Amount: Enter the loan amount (e.g., "450000")
  4. Property Address: Enter the property address
  5. Original Borrower: Enter the original borrower's name
  6. Click "Submit Application"
Tip: After submission, the application automatically enters the workflow and gets assigned to the first phase (INTAKE).

2 Understanding the Workflow Phases

Each loan application goes through 8 sequential phases. Each phase has a specialist responsible for completing specific tasks.

INTAKE APPLICATION DISCLOSURE LOAN REVIEW UNDERWRITING COMMITMENT CLOSING POST CLOSING

1 INTAKE

Initial document collection and verification. Gather all required documents from the borrower.

2 APPLICATION

Process the formal loan application. Verify borrower information and eligibility.

3 DISCLOSURE

Generate and send required disclosures. Ensure compliance with regulations.

4 LOAN REVIEW

Review loan terms and conditions. Verify all calculations are correct.

5 UNDERWRITING

Risk assessment and approval decision. Human decision required for final approval.

6 COMMITMENT

Issue commitment letter. Lock in loan terms and conditions.

7 CLOSING

Prepare closing documents. Schedule and conduct closing meeting.

8 POST CLOSING

Final document review. Transfer servicing and archive records.

4 Managing Specialist Allocation

The Specialist Allocation tab allows you to manage how specialists are assigned to different phases.

Allocation Buckets View

Each phase has a "bucket" showing assigned specialists. You can:

  1. Drag specialists between buckets to change their phase assignment
  2. Click the ↔ button on a specialist to reassign their tasks to another specialist in the same bucket
  3. View task counts for each specialist
  4. Manage dual-phase specialists who work across two phases

Task Reassignment Within Same Bucket

  1. Find a specialist with tasks (shows task count)
  2. Click the ↔ (swap) button next to their task count
  3. A panel opens showing all their tasks
  4. Use the dropdown on each task to select another specialist
  5. Task is instantly reassigned

Moving Specialists Between Phases

  1. Drag a specialist chip from one bucket to another
  2. If they have tasks, you'll see a Phase Transfer Modal
  3. Choose to either:
    • Reassign Tasks: Hand over tasks to other specialists
    • Go Dual-Phase: Specialist works in both phases
  4. Complete all required reassignments before confirming
Important: You cannot move the last specialist out of a bucket. At least one specialist must remain in each phase.

Directory Sub-Tab

View all specialists in a table format with:

  • Certificates (specialty types)
  • Current role and status
  • Assigned tasks (click to view details)
  • Edit and delete options

6 Working on Tasks in Specialist Workbench

The Specialist Workbench is where specialists complete their assigned tasks.

Workbench Layout

Task Queue

Left panel showing all tasks assigned to you. Click a task to work on it.

Task Details

Main area showing the selected task with all subtasks and checklist items.

Application Info

Side panel with borrower details, loan information, and status.

Completing a Task

  1. Select a task from your queue (left panel)
  2. Review the task details and requirements
  3. Work through each subtask:
    • Complete all checklist items
    • Add notes if needed
    • Mark subtask as complete
  4. Once all subtasks are done, click "Complete Task"
  5. The application moves to the next phase automatically
Tip: Use the notes feature to document any issues or special circumstances for the next specialist.

7 Underwriting Review & Decision

The Underwriting phase requires a human decision to approve or reject the loan application.

Accessing the Underwriting Review

  1. Login as underwriter (password: password123)
  2. Select an application in the UNDERWRITING or HUMAN_DECISION phase
  3. Click "Review Document & Decide" button

Review Modal Contents

The Underwriting Review Modal displays:

  • Borrower Information: Name, SSN (last 4)
  • Loan Details: Amount, property address, original borrower
  • Underwriting Checklist: Results from automated checks
  • Risk Assessment: Overall risk level and recommendation

Making a Decision

  1. Review all information in the document view
  2. Optionally add notes explaining your decision
  3. Click "Approve" (green) or "Reject" (red)
  4. If approved: Application moves to COMMITMENT phase
  5. If rejected: Application is marked as DENIED
